<h1>Tecboard</h1>

<p>Página inicial de apresentação da plataforma Tecboard, desenvolvida com HTML5 e CSS3.
O objetivo do projeto é apresentar uma interface moderna para um sistema de monitoramento de aplicações em tempo real,
  destacando informações importantes e incentivando o usuário a testar a versão demo.</p>
